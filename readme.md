Made this because as a new player I found it hard to find character kits for teambuilding.

This lets me filter for all awakeners that have vulnerable/poison/weakness/etc. mechanics.
The skill wording is inconsistent though so it's still hard to find but you can do regex filtering like `apply.*vulnerable`.

Feel free to add to this and update the JSON and make feature requests.

Will probably add more features later.
